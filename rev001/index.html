Estrutura da página

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar o conteúdo e a estrutura de uma página web.

Características principais

  • Usa tags para marcar e organizar o conteúdo (<h1>, <p>, <div>).
  • Define a estrutura, mas não cuida do estilo (isso é função do CSS).
  • A maioria das tags vem em pares: abertura <tag> e fechamento </tag>.

Semântica da página

HTML semântico é quando usamos tags que têm significado próprio, para indicar o papel do conteúdo na página.

Isso ajuda:

  • Leitores de tela (acessibilidade)
  • Motores de busca (SEO)
  • Outros desenvolvedores (código legível)
  • Você no futuro (manutenção mais fácil)

Por exemplo:

  • <div id="menu"> não diz nada sobre o que é o elemento.
  • <nav> informa que é um menu de navegação.

Principais tags semânticas

Tag Função Exemplo de uso
<header> Cabeçalho de página ou seção Logo, título, menu
<nav> Navegação principal ou secundária Menu de links

<mark> Texto destacado Palavras-chave
<time> Data ou hora <time datetime="2025-08-12">12/08/2025</time>
<address> Informações de contato Endereço, e-mail

Estilo da página

CSS (Cascading Style Sheets) é a linguagem de estilo que define a aparência da página (cores, fontes, espaçamento, layout...)

Características principais

  • Aplica o estilo ao HTML
  • Pode ser usado inline, interno ou externo (o mais comum é externo .css)
  • Usa seletores para escolher elementos (p, .classe, #id)

Formas de aplicação

A forma mais simples de aplicar estilos: CSS inline style

É a modificação pontual no código.

CSS internal style

Para aplicar estilos de forma mais dinâmica e prática, podemos adicionar uma tag <style> dentro da área <head> do nosso documento HTML local.

A técnica mais versátil: CSS external style

Manter as folhas de estilo fora do código HTML, além de uma maior organização, faz com que tudo seja reaproveitado de maneira mais eficiente nas outras páginas do site. Para isso, utilizamos a tag <link> especialmente configurada para trabalhar com arquivos externos de estilo. Essa tag deve ser colocada dentro da área <head> do seu documento HTML.

Qual utilizar?

A técnica CSS inline style deve ser evitada ao máximo. Ela acaba deixando seu código meio confuso, misturando a parte HTML e CSS em uma mesma linha. Já a técnica CSS internal style organiza melhor seu código, separando o conteúdo e estilo em duas áreas bem definidas do seu documento. Opte também por essa técnica apenas se a quantidade de configurações de estilo for pequena/média. Usar muitos seletores com muitas declarações vai fazer com que seu arquivo .html fique muito grande e seu conteúdo seja visualmente jogado lá pra baixo, dificultando manutenções futuras.

Por fim, opte sempre pela técnica CSS external style sempre que seu estilo for usado em várias páginas dentro do seu site. Usando a tag <link> em várias páginas, você pode compartilhar o mesmo estilo entre elas e não vai precisar ficar alterando vários arquivos quando o seu cliente solicitar uma pequena mudança no tom de uma determinada cor, por exemplo.

De forma resumida, guarde isso:

  • CSS externo: use sempre que puder.
  • CSS interno: use para pequenas configurações.
  • CSS inline: procure evitar.
